# Review System Web Application

A simple web-based review system built using HTML, CSS, and JavaScript.

## Files

- `review-system.html`: Main HTML file for the review system interface
- `styles.css`: CSS styles for the application
- `script.js`: JavaScript functionality for the review system

## Setup

1. Clone the repository
2. Open `review-system.html` in a web browser

## Features

- Add reviews
- Display reviews
- Basic styling

## Git Repository

This project is version controlled with Git and hosted on GitHub.
